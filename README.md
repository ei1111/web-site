# site-api

### 회원가입, 상품주문, 결재, 쿠폰 발행 프로젝트


## 기술 및 버전 정보

- **언어:** Java 21
- **프레임워크:** Spring Boot 3.5.6
- **빌드 도구:** Gradle
- **데이터베이스:** MySQL
- **ORM:** JPA
- **쿼리 빌더:** QueryDSL 5.0.0
- **로깅:** P6Spy
- **문서화:** Swagger (SpringDoc OpenAPI)

## 주요 기술 스택 및 설명

| 기술 | 설명 |
| --- | --- |
| **Spring Boot** | 빠른 서버 개발을 위한 프레임워크 |
| **JPA** | ORM 기반 DB 연동 |
| **QueryDSL** | 타입 안전한 SQL 작성 |
| **Gradle** | 빌드 및 의존성 관리 도구 |
| **JWT** | 인증 및 권한 처리 |
| **Swagger** | API 문서 자동화 (SpringDoc OpenAPI) |
| **P6Spy** | SQL 로그 시각화 및 디버깅 |
| **MySQL** | 관계형 데이터베이스 |
| **GitHub Actions** | CI/CD 자동화 워크플로우 |
